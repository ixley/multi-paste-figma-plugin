# Multi-Paste — Figma Plugin

Paste a set of values into a series of selected text elements.

---

## What it does

- Takes a line separated list and sequentially pastes each line of text to a selected text layer. Currently goes top to bottom, left to right.

---

## Installation

1. Clone or download this repository.
2. In Figma Desktop, go to **Plugins → Development → Import plugin from manifest…** and select `manifest.json`.

---

## Usage

Select a multiple text frames.

**Paster:** Opens UI to paste values and select options.

**Paste from Clipboard:** Prompts to paste directly from clipboard (skipping opening the window).
